# Features
- Auto check-in daily
- Auto send task

# Requirement
- VS Code
- Node.js or Bun

# How to run
- Open VS Code, open index.js
- Fill datas.txt with your private keys (each line)
- Open terminal
- Command:
  > npm install

  > node index
- Done
